# Docs

Run the following command in the docs directory

```bash
npm install http-server -g
http-server ./
```

The docs page will then open in the browser

# Examples

Run the following command in the examples directory

```bash
npm install
npm run dev
```

The examples page will then open in the browser

You can switch the link to see examples:

* http://localhost:8888/basic/index.html
* http://localhost:8888/miniViewer/index.html
